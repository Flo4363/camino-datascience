# Data Sources — Download & Reproduction Guide

This file documents every data file that should be present in `data/raw/`, `data/processed/`, and `data/external/`.
Raw data files are excluded from Git. Use this guide to reproduce the full dataset from scratch.

---

## data/raw/

### Oficina del Peregrino — Annual statistics (ES)
- **Files**: `YYYY_oficina_annual.pdf` for each year 2003–2025
- **Source**: https://oficinadelperegrino.com/estadisticas/
- **Download**: Manual PDF download, one file per year
- **Variables**: Compostelas issued, by route / nationality / sex / age group / means / motive / start point / month
- **Notes**: Spanish language. Tables extractable with `pdfplumber` or `tabula-py`.

### Bureau SJPDP — Saint-Jean-Pied-de-Port statistics (FR)
- **Files**: `YYYY_sjpdp_stats.pdf` or manually entered CSV
- **Source**: https://www.compostelle.fr (Association des Amis de Saint-Jacques)
- **Download**: Manual — annual publications, some years web only
- **Variables**: Monthly counts, nationality breakdown (partial years)

### Trail du Saint-Jacques — Finisher results
- **Files**: `YYYY_trail_sj_results.csv` (scraped from LiveTrail)
- **Source**: https://tsj.livetrail.net/
- **Download**: Scraping script in `src/scrape_livetrail.py` (to be created in NB01)
- **Coverage**: 2012–2024, editions: Ultra (79km), Grand Trail (51km), Maratrail, Chibottes

## data/external/

### ERA5 climate reanalysis
- **Files**: `era5_YYYY_MM_[variable].nc` (NetCDF format)
- **Source**: https://cds.climate.copernicus.eu (free account required)
- **Download**: `src/download_era5.py` using `cdsapi` library
- **Variables**: 2m temperature, total precipitation, wind speed
- **Grid points**: Defined per route corridor in `src/route_corridors.py`
- **Notes**: Files can be large (>100MB). Store locally, do not commit.

### Google Trends
- **Files**: `trends_KEYWORD_COUNTRY_YYYY.csv`
- **Source**: Google Trends via `pytrends` library
- **Download**: `src/download_trends.py` (to be created in NB01)
- **Keywords**: ["Camino de Santiago", "chemin de Compostelle", "Way of St James", "Jakobsweg", "Cammino di Santiago", "산티아고 순례길"]
- **Countries**: ES, FR, DE, IT, US, KR, BR, PT, GB (expand as needed)

### Media event database
- **File**: `media_events_manual.csv`
- **Source**: Manually curated
- **Construction**: See NB01 for methodology
- **Variables**: year, country, type (film/book/podcast/youtube_channel/trail_event), title, estimated_reach, notes
